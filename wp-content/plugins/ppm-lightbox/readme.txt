== Description ==

This plugin will enable lightbox in wordpress. Please read installation tab for more info.

### Plugin Features
* Easy to use.
* Very simple & lightweight (only 20KB).
* Youtube, Vimeo video auto embeding. 
* Gallery keyboard navigation & mobile touch supported. 
* Fully responsive.


Live Preview: http://perfectpointmarketing.com/plugins/ppm-lightbox




== Installation ==

1. Install as a regular WordPress plugin
2. This plugin is work via HTML rel attribute. You have to add rel attribute in link tag. 

### Examples:

= Single Image =

`<a href="YOUR_BIG_IMAGE_PATH" rel="prettyPhoto" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>`

= Image Gallery =

`
<div id="gallery">
	<a href="YOUR_BIG_IMAGE_PATH" rel="prettyPhoto[gallery1]" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>
	<a href="YOUR_BIG_IMAGE_PATH" rel="prettyPhoto[gallery1]" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>
	<a href="YOUR_BIG_IMAGE_PATH" rel="prettyPhoto[gallery1]" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>
	<a href="YOUR_BIG_IMAGE_PATH" rel="prettyPhoto[gallery1]" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>
	<a href="YOUR_BIG_IMAGE_PATH" rel="prettyPhoto[gallery1]" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>
</div>
`

= Youtube Video =

`<a href="YOUTUBE_VIDEO_LINK" rel="prettyPhoto" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>`

= Vimeo Video =

`<a href="VIMEO_VIDEO_LINK" rel="prettyPhoto" ><img src="YOUR_THUMBNAIL_PATH" alt=""/></a>`
